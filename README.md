# Snivalica-Mica
Ručno radjeni jastučići u obliku Maca, dinosaurusa, zeca, slona, zvezde, meseca, oblaka, sunca... od prirodnog materijala, pamuka i antialergijskog punjenja. Veselih boja  prikladni za malu i veliku decu. Održavaju se pranjem na 40 C ručno ili mašinski.
Zanimljivi bilo da se sa njima igrate, spavate ili dekorišete svoj životni prostor. Veseli drugari u domu i na putovanju.
